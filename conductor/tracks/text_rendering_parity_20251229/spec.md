# Track Spec: Text Rendering Parity Audit`n`n## Goal`nEnsure that text backgrounds, shadows, outlines, and hollow text render identically in the client-side Fabric.js editor and the server-side node-canvas generator.`n`n## Scope`n* **Text Backgrounds:** Verify color, padding, and corner radius.`n* **Text Shadows:** Verify blur, color, and offsets (X/Y).`n* **Text Outlines:** Verify stroke color and stroke width.`n* **Hollow Text:** Verify transparent fill with stroke rendering.`n`n## Success Criteria`n* A test suite or visual comparison tool that demonstrates identical output for all effects.`n* Fixes applied to any discrepancies found in the server-side rendering logic.
